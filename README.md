# DiceSimulation
Küçük bir zar atma projesi
# Getting started
gerekli kütüphaneleri indiiyoruz
```
pip install -r requirements.txt
```
daha sonra
```
python Dice.py --help
```
bu komut paramterleri gösteriyor
```
python Dice.py --trys <amount>
```
ya da
```
python Dice.py
```
bu şekilde kullanabilrisiniz.
